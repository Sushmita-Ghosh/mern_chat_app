## Getting Started

1. **Clone the repository:**
   ```bash
   git clone 
   cd mern_chat_app
   ```

2. **Install dependencies:**
 ```bash
   npm install
   ```

3. **Run the application:**
    ```bash
       npm start
   ```

5. **Open the application:**
  - Access the chat application at **http://localhost:3000** in your web browser.
